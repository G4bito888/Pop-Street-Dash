# Pop Street Dash

Welcome to **Pop Street Dash**! A fast-paced 2D runner with a vibrant *cyberfunk* aesthetic, cel-shaded art, and that heavy street feeling. Dodge obstacles, glide through the neon city, and keep the rhythm.

## About the Project
Developed as a high-speed game dev sprint, this project mixes the adrenaline of urban parkour with low-poly graphics and neon skater graffiti. It serves as a showcase of rapid prototyping, custom 2D asset creation, and seamless team collaboration.

## Basic Controls
* **Movement:** A/D Keys
* **Jump:** Spacebar
* **Glide:** Spacebar x2

## Tech Stack & Tools
* **Game Engine:** Unity
* **Language:** C#
* **Art & UI:** ibis Paint X
* **Audio Production:** GarageBand
* **Version Control:** Git & GitHub

## How to run locally
1. Clone this repository to your local machine.
2. Open **Unity Hub** and click on `Add project from disk`.
3. Select the *Pop-Street-Dash* root folder.
4. Open the `MainLevel` scene located in `Assets/Scenes` and hit the Play button!
